🧪 Prijzen via Google Shopping (SerpAPI)

Test of we via SerpAPI accurate marktprijzen kunnen ophalen voor wijnen.

🔧 Eenmalige setup

Voordat dit werkt, moet er een SerpAPI key in Railway staan:
  1. Maak een gratis account aan op serpapi.com (100 zoekopdrachten/maand gratis, geen creditcard)
  2. Kopieer je API key uit het dashboard
  3. Ga naar Railway → WineSteals project → Variables → voeg toe: SERPAPI_KEY = jouw key
  4. Wacht tot Railway opnieuw deploy't (~1 min), test dan hieronder
